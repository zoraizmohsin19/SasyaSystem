Platform

Cloud Based Data Store, Analytics and Mobile App

Work Process

Configure Device Data on Sasya Cloud

Sasya GoMos cloud app runs 24X7 securely to ingest high speed data from its thousands of sensors.

Configurations of devices, sensors, payloads, alerts, rules and trigger points can be done on GoMos.

Analytics & Ops Dashboard

Dashboards to see the real time sensor data, and last alerts generated.

Operating dashboard to see status of automated devices and manually operate devices

Notifications

Timely Alerts and Notifications with current reading of the parameters and possible advisory on next action.



Sasya Systems has the back-end platform ready which is currently being used in SaaS mode for its various clients. The platform provides rich interfaces that will be utilized in the project, such as -
  • API access is available on GoMos to get the raw or aggregated sensor data and create your actions and send it back to GoMos for Automation of devices
  • Configurations of devices, sensors, payloads, alerts, rules and trigger points.
  • Dashboards to show the real time sensor data, and last alerts generated.
  • Historic data is shown in graphs and table format. Data can be downloaded for offline analysis.
  • Operating dashboard provides a way to corelate sensor data with the status of the actuated machines (for e.g. water moisture sensor data correlation with the status of the pump being ON/OFF).
  • Configuration screens allow the set up of alerts, trigger points (max / min trigger) and screens to see the current status of these triggers.
  • Additionally, a web page will be provided for non-users (public domain) where content provided by PAU / World Bank will be uploaded for general public.
